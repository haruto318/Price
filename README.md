</div>

## 開発環境
- Swift  
- Xcode

## 使用API
- PriceAPI  
- Firebase

## Feature

- [x] 多様なオンラインストアの製品を閲覧可能
- [x] 同じ商品の異なるストアでの価格を比較
- [x] 実店舗での値段を表示
- [ ] 気に入った商品をカートに保存
